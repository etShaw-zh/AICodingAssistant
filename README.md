## NOTICE
⚠️本项目是一个基于GPT-3的编码助手,目前支持根据编码框架对MOOC论坛中话题下的回帖数据进行编码。  
当前项目还在开发中,以后会提供更多的功能,敬请期待。

## 目标用户
- **研究者**
  本项目可以帮助研究者快速对论坛中的回帖数据进行编码,提高编码效率,减少编码错误。
